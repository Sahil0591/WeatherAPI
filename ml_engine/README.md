# ML Engine

This package provides a lightweight, modular scaffolding for ML workflows used by WeatherAPI.

- ingestion: Data ingestion and basic validation.
- features: Feature engineering and transformations.
- training: Model training routines and baselines.
- evaluation: Model metrics and performance reporting.
- explainability: Model interpretation and feature-attribution utilities.
- artifacts: Persistence of models and related files.
- tests: Unit tests for package functionality.

Each subpackage includes typed function stubs with docstrings to guide implementation.
